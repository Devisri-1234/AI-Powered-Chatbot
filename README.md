# AI-Powered Chatbot

## Overview

AI-Powered Chatbot is a web-based chatbot application developed using Python, Flask, NLP, and SQLite. The chatbot can answer user queries, perform intent matching, and maintain chat history.

## Features

* Interactive chatbot interface
* NLP-based response generation
* Chat history storage
* User-friendly web interface
* SQLite database integration

## Technologies Used

* Python
* Flask
* Scikit-Learn
* SQLite
* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository

git clone https://github.com/yourusername/AI-Powered-Chatbot.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000
